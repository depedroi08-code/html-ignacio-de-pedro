Por qué empecé a programar

Autor: Ignacio • Publicado el 10 de marzo de 2025

Siempre me gustó la tecnología, pero durante mucho tiempo la veía como algo lejano: usaba aplicaciones, juegos y redes sociales sin entender qué pasaba por detrás. Todo cambió cuando descubrí que podía crear mis propias páginas web con unas pocas líneas de código.

Empecé con ejemplos muy simples de HTML y CSS, y de a poco fui ganando confianza. Lo que más me sorprendió fue la sensación de poder: escribir código, guardar el archivo y ver cómo el navegador mostraba algo completamente nuevo. A partir de ahí, me propuse seguir aprendiendo un poco cada día.

Herramientas que uso para estudiar programación

Autor: Ignacio • Publicado el 18 de marzo de 2025

Una de las claves para aprender a programar es organizar bien las herramientas que usamos. En mi caso, combino videos, cursos en línea, documentación oficial y, sobre todo, mucha práctica escribiendo código.

También me ayuda mucho usar un editor de texto cómodo, como Visual Studio Code, y tener un sistema de control de versiones con Git y GitHub. Esto no solo me permite guardar mis proyectos, sino también ver cómo van evolucionando con el tiempo.

Mi primera página en GitHub Pages

Autor: Ignacio • Publicado el 25 de marzo de 2025

Publicar una página en GitHub Pages fue un paso importante, porque significó compartir mi trabajo con otras personas. Al principio parecía complicado, pero en realidad se reduce a crear un repositorio, subir los archivos y activar la opción de Pages.

Lo más interesante es que, a partir de ese momento, cualquier cambio que hago en mis archivos HTML se puede ver en la web con solo actualizar el repositorio. Esto me motiva a seguir mejorando el diseño y el contenido de mis proyectos.

Consejos para no frustrarse aprendiendo a programar

Autor: Ignacio • Publicado el 2 de abril de 2025

Aprender a programar puede ser desafiante. Es normal encontrar errores, bugs y momentos donde nada parece funcionar. Lo importante es entender que el error forma parte del proceso de aprendizaje.

Algunos consejos que me sirvieron son: dividir los problemas en partes más pequeñas, pedir ayuda cuando me trabo, leer los mensajes de error con calma y, sobre todo, tener paciencia conmigo mismo. Cada línea de código que escribo, incluso si falla, me acerca un poco más a la solución.

Qué me gustaría aprender después de HTML y CSS

Autor: Ignacio • Publicado el 10 de abril de 2025

Después de aprender los fundamentos de HTML y CSS, el siguiente paso lógico es JavaScript, el lenguaje que le da interactividad a las páginas web. Con JavaScript puedo hacer que los sitios respondan a las acciones del usuario: botones, animaciones, formularios dinámicos y mucho más.

También me interesa explorar frameworks modernos como React o Vue, pero por ahora quiero consolidar bien las bases. Prefiero ir paso a paso, entendiendo lo que hago, antes de agregar más capas de complejidad.
